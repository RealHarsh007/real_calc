# 🧮 Calculator 電卓

A sleek and responsive calculator built using HTML, CSS, and JavaScript with a modern Neumorphism design. It supports basic arithmetic operations, percentage calculations, delete, and clear functions while providing a smooth user experience.

🔗 Live Demo: https://realharsh-calc.netlify.app/

## Tech Stack
- HTML5
- CSS3
- JavaScript

## Features
✔ Responsive Design  
✔ Neumorphism UI  
✔ Basic Arithmetic Operations  
✔ Percentage Calculation  
✔ AC & DEL Functions  
✔ Decimal Support
